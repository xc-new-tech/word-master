# 📚 WordMaster - 单词大师学习系统

基于维特根斯坦"语言游戏"哲学，结合艾宾浩斯记忆曲线的智能单词学习系统。

## ✨ 项目特点

- 🎯 **智能学习路径** - 4种学习模式：顺序学习、随机探索、智能推荐、考前冲刺
- 🎮 **语言游戏理念** - 每个单词都有独特的"语言游戏"描述，帮助理解本质
- 📊 **数据可视化** - 完整的学习统计、进度追踪、热力图展示
- 🌓 **深色模式** - 完美支持深色/浅色主题切换
- 📱 **响应式设计** - 适配手机、平板、桌面端
- 💾 **本地存储** - 学习数据持久化保存
- 🎨 **精美UI** - 基于Material Design，使用Tailwind CSS

## 💡 核心概念

### 语言游戏（Wittgenstein）

每个单词都包含"语言游戏"描述，揭示单词在真实语境中的使用本质：

- `negotiate` → "在利益拉锯战中寻找双赢点"
- `resilient` → "描述打不倒的小强精神"
- `ephemeral` → "捕捉并感叹那些转瞬即逝的美好"

### 双棋盘系统

- **思辨场**（学术/正式语境）：学术论文、正式演讲、考试作文
- **生活场**（日常/口语语境）：日常对话、社交媒体、朋友聊天

### 记忆技巧

- 词根拆解可视化
- 记忆故事与联想
- 易混淆词辨析
- 高频搭配
